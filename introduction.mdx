# Welcome to KoroFood's Restaurant Management System

KoroFood's is a comprehensive event and restaurant management system built with a modern microservices architecture. It provides a complete solution for managing reservations, events, tables, orders, and payments with real-time capabilities and multimedia integration.

## Architecture Overview

The system is built on a microservices architecture where each service is independent, making it resilient to change and highly scalable. Communication between services is handled via RabbitMQ for asynchronous messaging and WebSocket for real-time updates.

### Core Services

- **User Service** (Port 8081) - Authentication, user management, and AI chatbot
- **Table Service** (Port 8082) - Table management and availability
- **Reservation Service** (Port 8083) - Reservation booking and queries
- **Qualification Service** (Port 8084) - Ratings and reviews
- **Payment Service** (Port 8085) - Payment processing and validation
- **Order Service** (Port 8086) - Order management and reporting
- **Menu Service** (Port 8087) - Menu and dish management
- **Event Service** (Port 8088) - Event scheduling and themes
- **API Gateway** (Port 8098) - Single entry point for all requests
- **Eureka Server** (Port 8761) - Service discovery and registration

## Technology Stack

### Backend
- **Spring Boot 3.3.x** - Main framework for microservices
- **Java 21** - Programming language
- **RabbitMQ** - Asynchronous messaging
- **PostgreSQL** - Primary relational database
- **MongoDB** - Chat message storage
- **Redis** - High-performance caching
- **JWT & OAuth 2.0** - Authentication and authorization

### Frontend
- **Angular 17+** - User interface framework
- **TypeScript** - Type-safe development
- **WebSockets** - Real-time communication

### Infrastructure
- **Docker** - Containerization
- **Kubernetes** - Container orchestration
- **Cloudinary** - Image management
- **Eureka** - Service discovery
- **Spring Cloud Gateway** - API Gateway
