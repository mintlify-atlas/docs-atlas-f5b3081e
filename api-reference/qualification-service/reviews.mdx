## List All Reviews

<api-endpoint method="GET" endpoint="/calificacion" />

Retrieve all reviews and ratings in the system.

### Response

<ResponseField name="valor" type="boolean">
  Success indicator
</ResponseField>

<ResponseField name="mensaje" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of review objects
  <ResponseField name="idResena" type="integer">
    Review ID
  </ResponseField>
  <ResponseField name="idUsuario" type="integer">
    User ID who submitted the review
  </ResponseField>
  <ResponseField name="nombreUsuarioCompleto" type="string">
    Full name of the user
  </ResponseField>
  <ResponseField name="imagenUsuario" type="string">
    User profile image URL
  </ResponseField>
  <ResponseField name="idEntidad" type="integer">
    ID of the rated entity
  </ResponseField>
  <ResponseField name="nombreEntidad" type="string">
    Name of the rated entity
  </ResponseField>
  <ResponseField name="imagenEntidad" type="string">
    Entity image URL
  </ResponseField>
  <ResponseField name="calificacion" type="integer">
    Rating score (1-5)
  </ResponseField>
  <ResponseField name="comentario" type="string">
    Review comment
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.korofood.com/calificacion
```
</RequestExample>

## Get Reviews by User

<api-endpoint method="GET" endpoint="/calificacion/usuario/{id}" />

Retrieve all reviews submitted by a specific user.

### Path Parameters

<ParamField path="id" type="integer" required>
  User ID
</ParamField>

### Response

<ResponseField name="valor" type="boolean">
  Success indicator
</ResponseField>

<ResponseField name="mensaje" type="string">
  Response message
</ResponseField>

<ResponseField name="data" type="array">
  Array of review objects submitted by the user
</ResponseField>

<RequestExample>

curl https://api.korofood.com/calificacion/usuario/123
