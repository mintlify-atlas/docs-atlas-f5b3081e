## Create Dish

<api-endpoint method="POST" endpoint="/platos" />

Create a new dish in the menu.

### Request Body

<ParamField body="nombre" type="string" required>
  Dish name (max 100 characters)
</ParamField>

<ParamField body="precio" type="number" required>
  Dish price (must be positive)
</ParamField>

<ParamField body="stock" type="integer" required>
  Available stock (cannot be negative)
</ParamField>

<ParamField body="tipoPlato" type="string" required>
  Dish type: "E" (Entrada/Appetizer), "S" (Segundo/Main), "P" (Postre/Dessert), "B" (Bebida/Beverage)
</ParamField>

<ParamField body="imagen" type="string">
  Image URL for the dish
</ParamField>

<ParamField body="imagenBase64" type="string">
  Base64 encoded image data
</ParamField>

### Response

<ResponseField name="idPlato" type="integer">
  Dish ID
</ResponseField>

<ResponseField name="nombre" type="string">
  Dish name
</ResponseField>

<ResponseField name="precio" type="number">
  Dish price
</ResponseField>

<ResponseField name="stock" type="integer">
  Available stock
</ResponseField>

<ResponseField name="tipoPlato" type="string">
  Dish type code
</ResponseField>

<ResponseField name="tipoPlayoDescripcion" type="string">
  Dish type description
</ResponseField>

<ResponseField name="imagen" type="string">
  Image URL
</ResponseField>

<ResponseField name="activo" type="boolean">
  Whether the dish is active
</ResponseField>

<ResponseField name="etiquetas" type="array">
  Array of tag objects associated with the dish
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.korofood.com/platos \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Ceviche de Pescado",
    "precio": 28.50,
    "stock": 50,
    "tipoPlato": "E",
    "imagen": "https://example.com/ceviche.jpg"
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "idPlato": 101,
  "nombre": "Ceviche de Pescado",
  "precio": 28.50,
  "stock": 50,
  "tipoPlato": "E",
  "tipoPlayoDescripcion": "Entrada",
  "imagen": "https://example.com/ceviche.jpg",
  "activo": true,
  "etiquetas": []
}
```
</ResponseExample>

## Delete Dish

<api-endpoint method="DELETE" endpoint="/platos/{id}" />

Delete a dish (soft delete - marks as inactive).

### Path Parameters

<ParamField path="id" type="integer" required>
  Dish ID to delete
</ParamField>

### Response

Returns HTTP 204 No Content on success.

<RequestExample>
```bash cURL
curl -X DELETE https://api.korofood.com/platos/101
```
</RequestExample>

<ResponseExample>
```text
HTTP/1.1 204 No Content
```
</ResponseExample>

## Generate Dishes Report

<api-endpoint method="GET" endpoint="/platos/reporte/platos" />

Generate a PDF report of all dishes.

### Response

Returns a PDF file with the dishes report.

<RequestExample>
```bash cURL
curl https://api.korofood.com/platos/reporte/platos \
  -o reporte-platos.pdf
```
</RequestExample>

<ResponseExample>
```text
PDF file downloaded successfully.
Filename: reporte-platos.pdf
```
</ResponseExample>
