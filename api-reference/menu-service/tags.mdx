## Create Tag

<api-endpoint method="POST" endpoint="/etiquetas" />

Create a new tag for categorizing dishes.

### Request Body

<ParamField body="nombre" type="string" required>
  Tag name (max 100 characters)
</ParamField>

<ParamField body="descripcion" type="string">
  Tag description (max 500 characters)
</ParamField>

### Response

<ResponseField name="idEtiqueta" type="integer">
  Tag ID
</ResponseField>

<ResponseField name="nombre" type="string">
  Tag name
</ResponseField>

<ResponseField name="descripcion" type="string">
  Tag description
</ResponseField>

<ResponseField name="activo" type="boolean">
  Whether the tag is active
</ResponseField>

<ResponseField name="fechaRegistro" type="string">
  Registration date and time
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.korofood.com/etiquetas \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Vegetariano",
    "descripcion": "Platos sin carne"
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "idEtiqueta": 5,
  "nombre": "Vegetariano",
  "descripcion": "Platos sin carne",
  "activo": true,
  "fechaRegistro": "2026-03-12T10:30:00"
}
```
</ResponseExample>

## Delete Tag

<api-endpoint method="DELETE" endpoint="/etiquetas/{id}" />

Delete a tag (soft delete - marks as inactive).

### Path Parameters

<ParamField path="id" type="integer" required>
  Tag ID to delete
</ParamField>

### Response

Returns HTTP 204 No Content on success.

<RequestExample>
```bash cURL
curl -X DELETE https://api.korofood.com/etiquetas/1
```
</RequestExample>

<ResponseExample>
```text
HTTP/1.1 204 No Content
```
</ResponseExample>
