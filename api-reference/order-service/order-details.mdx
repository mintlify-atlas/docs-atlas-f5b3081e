## Get Order Details

<api method="GET" endpoint="/pedido/list/{pedidoId}">
  Retrieve all items (details) for a specific order.
</api>

### Path Parameters

<ParamField path="pedidoId" type="Integer" required>
  Order ID to retrieve details for
</ParamField>

### Response

<ResponseField name="valor" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="data" type="List<DetallePedidoResponse>">
  List of order detail items
  
  <ResponseField name="idDetalle" type="Integer">
    Detail ID
  </ResponseField>
  
  <ResponseField name="nombrePlato" type="String">
    Dish name
  </ResponseField>
  
  <ResponseField name="cantidad" type="Integer">
    Quantity ordered
  </ResponseField>
  
  <ResponseField name="precioUnitario" type="Double">
    Unit price
  </ResponseField>
  
  <ResponseField name="subtotal" type="Double">
    Line item subtotal
  </ResponseField>
  
  <ResponseField name="estado" type="String">
    Item status
  </ResponseField>
  
  <ResponseField name="observaciones" type="String">
    Special instructions
  </ResponseField>
</ResponseField>

<ResponseField name="mensaje" type="String">
  Response message
</ResponseField>

### Example Request

```bash
curl -X GET "https://api.korofood.com/pedido/list/456" \
  -H "Content-Type: application/json"
```

### Example Response

```json
{
  "valor": true,
  "data": [
    {
      "idDetalle": 789,
      "nombrePlato": "Ceviche de Pescado",
      "cantidad": 2,
      "precioUnitario": 15.99,
      "subtotal": 31.98,
      "estado": "PENDIENTE",
      "observaciones": "Sin cebolla"
    },
    {
      "idDetalle": 790,
      "nombrePlato": "Lomo Saltado",
      "cantidad": 1,
      "precioUnitario": 18.50,
      "subtotal": 18.50,
      "estado": "EN_PREPARACION",
      "observaciones": null
    }
  ],
  "mensaje": "Detalles obtenidos exitosamente"
}
```

## Mark Order as Paid

<api method="PUT" endpoint="/pedido/estadoPagado">
  Mark an order as paid after payment has been processed.
</api>

### Query Parameters

<ParamField query="idPedido" type="Integer" required>
  Order ID to mark as paid
</ParamField>

### Response

<ResponseField name="valor" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="data" type="Pedido">
  Updated order object
  
  <ResponseField name="idPedido" type="Integer">
    Order ID
  </ResponseField>
  
  <ResponseField name="estado" type="EstadoPedido">
    Updated order status (PAGADO)
  </ResponseField>
  
  <ResponseField name="fechaPago" type="LocalDateTime">
    Payment timestamp
  </ResponseField>
</ResponseField>

<ResponseField name="mensaje" type="String">
  Response message
</ResponseField>

### Example Request

```bash
curl -X PUT "https://api.korofood.com/pedido/estadoPagado?idPedido=456" \
  -H "Content-Type: application/json"
```

### Example Response

```json
{
  "valor": true,
  "data": {
    "idPedido": 456,
    "estado": "PAGADO",
    "fechaPago": "2026-03-12T15:45:00",

}
```
